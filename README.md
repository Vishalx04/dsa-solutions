# DSA Solutions

A collection of my solutions to popular **Data Structures & Algorithms (DSA)** problems.

## Problems

### 1. 3Sum
Find all unique triplets in an array that sum to zero.  
[LeetCode Problem Link](https://leetcode.com/problems/3sum/)

### 2. 4Sum
Find all unique quadruplets in an array that sum to a target value.  
[LeetCode Problem Link](https://leetcode.com/problems/4sum/)

### 3. Largest Subarray with Sum 0
Find the length of the largest subarray whose sum is 0.  
[TakeUForward Problem Link](https://takeuforward.org/plus/dsa/problems/largest-subarray-with-sum-0)

### 4. Count subarrays with given xor K
Find all unique quadruplets in an array that sum to a target value.  
[TakeUForward Problem Link](https://takeuforward.org/plus/dsa/problems/count-subarrays-with-given-xor-k)
